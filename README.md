# 3. 安装方式
目前，安装方式有以下几种：


## 3.1. 方式1：通过 npm 安装
```
npm install package-tls -D
```

## 3.2. 方式2：通过 pnpm 安装
```
pnpm add package-tls -D
```


## 3.3. 方式3：通过 yarn 安装
```
yarn add package-tls -D
```
